 Todo List

A simple and responsive Todo List web application built with HTML, CSS, and Vanilla JavaScript. The application allows users to manage tasks efficiently with features such as task creation, editing, completion tracking, filtering, and persistent local storage.

Features

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Filter tasks by:

  * All
  * Active
  * Completed
* Persistent data storage using Local Storage
* Animated buttons with smooth transitions
* Dreamy galaxy-themed background
* Responsive and modern glassmorphism UI

 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage API

Project Structure

 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/galaxy-todo-list.git
```

2. Navigate to the project folder:

```bash
cd todo-list
```

3. Open `index.html` in your browser.

No additional dependencies or installation steps are required.

 Usage

 Add a Task

Enter a task in the input field and click the "Add" button or press Enter.

 Complete a Task

Click the ✓ button to toggle the task's completion status.

 Edit a Task

Click the ✏ button to modify the task text.

 Delete a Task

Click the 🗑 button to permanently remove the task.

 Filter Tasks

Use the filter buttons to view:

* All Tasks
* Active Tasks
* Completed Tasks

 Data Persistence

Tasks are automatically saved in the browser's Local Storage and remain available after page refreshes or browser restarts.
